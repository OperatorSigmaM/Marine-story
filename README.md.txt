# История Марин

Этот репозиторий содержит статическую HTML-страницу с рассказом Марин — текст, который живёт на границе между словом и тишиной.  
Минимализм, чистый код и немного дыхания ветра.

## Просмотр
Сайт доступен через GitHub Pages по адресу:  
https://operatorsigmam.github.io/Marine-story/

## Структура
- `index.html` — основная страница с рассказом.


## Локальный просмотр
Просто открой `index.html` в браузере.

## Лицензия
Свободное использование с сохранением авторства.
